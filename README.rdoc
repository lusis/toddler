= toddler

Toddler is a simulation implemented in pure Ruby.

== Usage
	>> require 'toddler'
	=> true
	>> gus = Toddler.new
	>> gus.do_you_want_some_dinner?
	No
	>> gus.go_to_bed!
	No
	>> gus.please_go_to_bed
	No
	>> gus.please_go_to_bed
	No
	>> gus.please_go_to_bed
	No
	>> gus.please_go_to_bed
	No
	Yes
